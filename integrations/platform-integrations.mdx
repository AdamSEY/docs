## Marketing Platform Integrations

### Quick Setup (All Platforms)

All marketing platforms follow the same basic setup:

<Steps>
  <Step title="Get API Key">
    Get API key from your platform:
    - **Mailchimp:** Account ‚Üí Extras ‚Üí API keys
    - **Klaviyo:** Account ‚Üí Settings ‚Üí API Keys (need private key)
    - **HubSpot:** Settings ‚Üí Integrations ‚Üí API Key
  </Step>

  <Step title="Connect in Migma">
    Settings ‚Üí Integrations ‚Üí Marketing Platforms ‚Üí Connect ‚Üí Paste API key
  </Step>

  <Step title="Configure">
    Select audiences/lists to sync and map custom fields
  </Step>
</Steps>

### Mailchimp

**What you can do:**
- Export emails as Mailchimp campaigns or templates
- Sync subscriber lists bidirectionally
- Use Mailchimp merge tags: `*|FNAME|*`, `*|EMAIL|*`, `*|UNSUB|*`

**Export:** Create email in Migma ‚Üí Export ‚Üí Mailchimp Campaign ‚Üí Configure ‚Üí Push to Mailchimp as draft

### Klaviyo

**What you can do:**
- Export templates for campaigns and flows
- Sync lists and segments
- Use Klaviyo Liquid variables: `{{ first_name }}`, `{{ person.company }}`

**Export:** Create email ‚Üí Export ‚Üí Klaviyo Template ‚Üí Use in campaigns or automated flows

### HubSpot

**What you can do:**
- Export email templates
- Sync contact lists
- Use HubL tokens: `{{ contact.firstname }}`, `{{ contact.company }}`

**Setup:** OAuth connection via Settings ‚Üí Integrations ‚Üí Marketing Platforms

## Email Service Provider (ESP) Integrations

### Quick Comparison

| ESP | Best For | Free Tier | Setup Time |
|-----|----------|-----------|------------|
| **Amazon SES** | High volume, low cost | 62k/mo | 15 min |
| **SendGrid** | Analytics, deliverability | 100/day | 5 min |
| **Resend** | Developers, React Email | 3k/mo | 3 min |

### Amazon SES

**Best for:** High-volume sending, transactional emails, cost-effective ($0.10 per 1,000 emails)

**Quick setup:**
1. Create AWS account
2. Verify domain in SES Console (add DNS records)
3. Request production access (24hr approval)
4. Create IAM user with SES permissions
5. In Migma: Settings ‚Üí Integrations ‚Üí Amazon SES ‚Üí Add credentials

<Card title="Detailed SES Setup" icon="aws" href="https://docs.aws.amazon.com/ses/">
  Full AWS SES documentation
</Card>

### SendGrid

**Best for:** Detailed analytics, marketing + transactional

**Quick setup:**
1. Sign up at sendgrid.com
2. Verify sender domain (Settings ‚Üí Sender Authentication)
3. Create API Key (Settings ‚Üí API Keys)
4. In Migma: Settings ‚Üí Integrations ‚Üí SendGrid ‚Üí Paste API key

**Features:** Email validation, bounce tracking, click tracking, open tracking

### Resend

**Best for:** Developers, modern API, React Email support

**Quick setup:**
1. Sign up at resend.com
2. Add and verify domain (add DNS records)
3. Create API Key
4. In Migma: Settings ‚Üí Integrations ‚Üí Resend ‚Üí Paste API key

**Perfect combo:** Migma generates React Email ‚Üí Resend sends it natively

## Best Practices

<AccordionGroup>
  <Accordion title="Domain Authentication" icon="shield-check">
    **Always authenticate your domain:**

    Set up SPF, DKIM, and DMARC records for better deliverability.

    All ESPs provide DNS records to add. This prevents spoofing and improves inbox placement.
  </Accordion>

  <Accordion title="Start Slow" icon="gauge">
    **Warm up your sending domain:**

    - Day 1: 50 emails
    - Day 2: 100 emails
    - Day 3: 200 emails
    - Gradually increase

    Establishes sender reputation and avoids spam filters.
  </Accordion>

  <Accordion title="Monitor Metrics" icon="chart-line">
    **Track key indicators:**

    - **Delivery rate:** Should be greater than 99%
    - **Open rate:** Industry average 15-25%
    - **Click rate:** Industry average 2-5%
    - **Bounce rate:** Should be less than 2%
    - **Complaint rate:** Should be less than 0.1%
  </Accordion>

  <Accordion title="List Hygiene" icon="broom">
    **Keep lists clean:**

    - Remove hard bounces immediately
    - Remove soft bounces after 3 attempts
    - Honor unsubscribes instantly
    - Validate emails before adding

    Better deliverability, lower costs, maintain sender reputation.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection failed" icon="link-slash">
    **Problem:** Can't connect ESP to Migma

    **Solutions:**
    - Verify API key is correct and active
    - Check API key has proper permissions
    - Ensure domain is verified in ESP
    - Test credentials in ESP platform first
  </Accordion>

  <Accordion title="Emails not delivering" icon="envelope-circle-check">
    **Problem:** Emails sent but not received

    **Check:**
    - Domain authentication (SPF, DKIM, DMARC)
    - ESP bounce logs
    - Recipient's spam folder
    - Email content for spam triggers
    - Sending rate limits
  </Accordion>

  <Accordion title="Variables not replacing" icon="bracket-curly">
    **Problem:** Merge tags showing as `{{first_name}}` instead of actual names

    **Solutions:**
    - Check variable syntax matches ESP (Mailchimp: `*|FNAME|*`, Klaviyo: `{{ first_name }}`)
    - Verify fields mapped correctly
    - Ensure subscriber data has those fields
  </Accordion>
</AccordionGroup>
