## Popular Use Cases

### Marketing Automation
```javascript
// Generate a promotional email
const response = await fetch('https://api.migma.ai/v1/projects/{projectId}/conversations', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: "Create a summer sale announcement with 30% off, highlighting our new collection",
    purpose: "promotional"
  })
});
```

### E-commerce Integration
```javascript
// Generate product announcement
const response = await fetch('https://api.migma.ai/v1/projects/{projectId}/conversations', {
```
