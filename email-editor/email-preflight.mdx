## Common Issues & Fixes

<AccordionGroup>
  <Accordion title="üî¥ Dark Mode Text Unreadable" icon="moon">
    **Problem:** White text on white background in dark mode

    **Fix:**
    ```css
    /* Add to your email CSS */
    @media (prefers-color-scheme: dark) {
      .dark-safe-text {
        color: #000000 !important;
      }
    }
    ```

    Or tell Migma: "Fix dark mode text visibility"
  </Accordion>

  <Accordion title="üî¥ Broken Links" icon="link-slash">
    **Problem:** Links return 404 or errors

    **Fix:**
    1. Review broken link list
    2. Update URLs in email
    3. Re-run preflight to verify
  </Accordion>

  <Accordion title="‚ö†Ô∏è  High Spam Score" icon="shield-exclamation">
    **Problem:** Spam score above 5/10

    **Common causes:**
    - Too many exclamation marks!!!!
    - ALL CAPS WORDS
    - Spam trigger words (FREE, URGENT, WINNER)
    - Too many links
    - No unsubscribe link

    **Fix:**
    - Remove excessive punctuation
    - Use sentence case
    - Reduce link density
    - Add unsubscribe link
  </Accordion>

  <Accordion title="‚ö†Ô∏è  Missing Alt Text" icon="image-slash">
    **Problem:** Images without alt text

    **Fix:**
    1. Click each image
    2. Add descriptive alt text
    3. Benefits: accessibility + shows if images blocked
  </Accordion>

  <Accordion title="üî¥ Layout Breaks on Outlook" icon="envelope-open-text">
    **Problem:** Email looks broken in Outlook 2016/2019

    **Cause:** Outlook uses Word rendering engine (limited CSS)

    **Fix:**
    - Use table-based layouts
    - Avoid flexbox/grid
    - Use inline styles
    - Tell Migma: "Make compatible with Outlook"
  </Accordion>

  <Accordion title="‚ö†Ô∏è  Mobile Text Too Small" icon="text-height">
    **Problem:** Text under 14px on mobile

    **Fix:**
    - Increase font sizes for mobile
    - Use media queries
    - Or: "Increase mobile text sizes"
  </Accordion>
</AccordionGroup>
