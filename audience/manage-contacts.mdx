## Viewing Contacts

Navigate to `/contacts` in your project to view all subscribers in a sortable, filterable table.

**Table columns:**
- Email address (sortable)
- Name (sortable)
- Tags (colored badges, clickable to filter)
- Status badge (subscribed, unsubscribed, non-subscribed, bounced)
- Date added (sortable)
- Actions menu (Edit, Send Email, Delete)

**Quick actions:**
- **Add Subscriber** - Add single contact
- **Bulk Import** - Upload CSV file
- **Search** - Find by email or name
- **Filter** - By status and tags
- **Sort** - By email, name, or date

---

## Search & Filter

### Search

Use the search box to find contacts by email or name:

```
john@example.com    → Find exact email
@gmail.com          → All Gmail subscribers
John                → Names or emails containing "John"
```

### Filter by Status

<Tabs>
  <Tab title="Subscribed">
    **Active subscribers** - Can receive marketing emails

    Set via: Signup forms, CSV import, manual add
  </Tab>

  <Tab title="Unsubscribed">
    **Opted out** - Cannot receive marketing emails

    Set via: Unsubscribe link, preference center, manual change

    <Warning>CSV import respects unsubscribed status - won't re-subscribe</Warning>
  </Tab>

  <Tab title="Non-subscribed">
    **Not opted in** - Added to database but not subscribed

    Use for: Imported lists without consent, require explicit opt-in
  </Tab>

  <Tab title="Bounced">
    **Invalid email** - Delivery failed, cannot receive emails

    Causes: Invalid format, non-existent domain, disabled mailbox
  </Tab>
</Tabs>

### Filter by Tags

- Click tag badges to filter by that tag
- Select multiple tags from dropdown (OR logic)
- Clear individual tags or reset all filters

## Adding Contacts

<Tabs>
  <Tab title="Add Single Contact">
    **Manual add:**

    1. Click **"Add Subscriber"**
    2. Enter email (required)
    3. Add name, tags, custom fields (optional)
    4. Set status (default: subscribed)
    5. Click **"Add Contact"**

    **Duplicate handling:** If email exists, shows error with option to view existing contact
  </Tab>

  <Tab title="Bulk CSV Import">
    **Import multiple contacts:**

    <Card title="CSV Upload Guide" icon="file-csv" href="/audience/csv-upload">
      Complete guide to bulk importing contacts via CSV with field mapping
    </Card>

    **Quick summary:**
    - Upload CSV with email column (required)
    - Map columns to fields (email, name, tags, custom fields)
    - Set default status for new contacts
    - Duplicates are automatically skipped
  </Tab>
</Tabs>

## Custom Fields

Store additional data for each contact using key-value pairs.

**Common use cases:**
- Company name
- Job title
- Purchase history
- User preferences
- Account IDs
- Custom attributes

**Adding custom fields:**

<Steps>
  <Step title="During Add/Edit">
    Click **"Add Custom Field"** when adding or editing a contact
  </Step>
  <Step title="Enter Key-Value">
    - Key: `company`
    - Value: `Acme Corp`
  </Step>
  <Step title="Save">
    Custom fields are stored with the contact
  </Step>
</Steps>

**Using in emails:**
- Reference in email templates with merge tags
- Segment by custom field values
- Personalize content based on data

## Tags

Organize contacts with tags for segmentation and filtering.

**Tag features:**
- Multiple tags per contact
- Color-coded badges
- Click to filter
- Reusable across contacts

**Example tags:**
- `VIP` - High-value customers
- `Newsletter` - Newsletter subscribers
- `Trial` - Trial users
- `Q4-2024` - Campaign-specific

**Managing tags:**
- Add during contact creation
- Add/remove when editing
- Filter by one or multiple tags
- Tags auto-complete from existing tags

## Deleting Contacts

**Delete individual contact:**
1. Click Actions menu → **Delete**
2. Confirm deletion
3. Contact permanently removed

**Bulk delete:**
- Filter/search for contacts to delete
- Select multiple (future feature)

<Warning>
**Permanent deletion** - Cannot be undone. Consider tagging as inactive instead of deleting.
</Warning>

## Best Practices

<AccordionGroup>
  <Accordion title="Maintain List Hygiene" icon="broom">
    - Remove bounced emails promptly
    - Honor unsubscribes immediately
    - Archive inactive subscribers (6+ months no opens)
    - Never re-subscribe unsubscribed contacts
  </Accordion>

  <Accordion title="Use Tags Strategically" icon="tags">
    - Create consistent tag naming convention
    - Tag by: interest, engagement, source, campaign
    - Remove outdated tags regularly
    - Use tags for segmentation
  </Accordion>

  <Accordion title="Leverage Custom Fields" icon="database">
    - Store data you'll use for personalization
    - Keep keys consistent (e.g., `company`, not `Company Name` or `comp`)
    - Don't store sensitive data (passwords, payment info)
    - Use for segmentation and targeting
  </Accordion>

  <Accordion title="Regular Maintenance" icon="wrench">
    - Review contact list monthly
    - Clean up bounced emails
    - Update tags and custom fields
    - Export backups regularly
  </Accordion>
</AccordionGroup>
