## Method 1: Automatic Brand Import (Recommended)

The fastest way to set up your brand is to let Migma extract everything automatically from your website.

### What Gets Imported

<AccordionGroup>
  <Accordion title="Logos" icon="image">
    **Migma extracts:**
    - Primary logo (from header/homepage)
    - Secondary logo variations
    - Favicon

    **Supported formats:** SVG, PNG, JPG, WebP

    <Info>
    Migma intelligently finds the highest quality versions and removes backgrounds automatically.
    </Info>
  </Accordion>

  <Accordion title="Color Palette" icon="palette">
    **Migma detects:**
    - Primary brand color
    - Secondary/accent colors
    - Full color palette (up to 10 colors)
    - Color usage patterns

    **How it works:** AI analyzes your website's CSS, images, and visual hierarchy to identify your core brand colors.
  </Accordion>

  <Accordion title="Typography" icon="font">
    **Migma identifies:**
    - Heading font family
    - Body text font family
    - Font weights and styles
    - Font fallbacks

    **Google Fonts supported:** Migma automatically includes Google Fonts in email-safe format.
  </Accordion>

  <Accordion title="Brand Voice & Tone" icon="microphone">
    **Migma analyzes:**
    - Website copy and messaging
    - Tone and personality
    - Brand values and positioning
    - Writing style patterns

    **Result:** AI learns how to write in your brand's voice for email copy.
  </Accordion>

  <Accordion title="Visual Style" icon="paintbrush">
    **Migma captures:**
    - Design patterns and layouts
    - Button styles
    - Spacing and typography scale
    - Component aesthetics

    **Usage:** Informs email layout and design choices.
  </Accordion>
</AccordionGroup>

    <Tip>

    </Tip>
  </Step>

  <Step title="Start Import">
    Click **"Import Brand"** to begin automatic extraction
  </Step>

  <Step title="Wait for Processing">
    Import typically takes 30-60 seconds. You'll see progress indicators for each step:
    - üîç Analyzing website...

  </Step>

    Fine-tune any elements that need correction (optional)
  </Step>

  <Step title="Save Project">
    Click **"Save Project"** to finalize your brand configuration
  </Step>
</Steps>

<Warning>
**Import Failed?** If import fails:
- Ensure your website is publicly accessible
- Try your homepage instead of a subpage
- Check that your website doesn't block crawlers
- Use manual configuration as an alternative
</Warning>

Upload your logo files to use in email headers and footers.

### Define Color Palette

Set your brand colors that will be used throughout your emails.

**Color Configuration:**

<Tabs>
  <Tab title="Primary Color">
    Your main brand color, used for:
    - CTA buttons
    - Links
    - Headers and accents
    - Important elements

    **Tip:** Choose your most recognizable brand color
  </Tab>

  <Tab title="Secondary Color">
    Your accent color, used for:
    - Secondary buttons
    - Highlights
    - Icons and badges
    - Complementary elements

    **Tip:** Should complement your primary color
  </Tab>

  <Tab title="Additional Colors">
    Extended palette for more variety:
    - Background colors
    - Text variations
    - Section dividers
    - Visual interest

    **Tip:** Add 3-5 additional colors from your brand palette
  </Tab>
</Tabs>

**How to Add Colors:**

1. Click on the color picker
2. Enter hex code (e.g., `#6D28D9`) or use the visual picker
3. Name your color (e.g., "Brand Purple", "Accent Green")
4. Add additional colors as needed

<Note>
**Accessibility:** Migma automatically checks color contrast ratios and warns if text readability might be an issue.
</Note>

### Configure Typography

Choose fonts that match your brand identity.

**Font Selection:**

<AccordionGroup>
  <Accordion title="Heading Font" icon="heading">
    Used for email subject lines, h1, h2, h3 headings

    **Options:**
    - Google Fonts library (500+ fonts)
    - Web-safe fallback fonts
    - Custom font URLs (advanced)

    **Popular choices:** Montserrat, Poppins, Playfair Display, Roboto, Inter

    <Tip>
    Choose a font that's bold and readable at various sizes
    </Tip>
  </Accordion>

  <Accordion title="Body Font" icon="align-left">
    Used for paragraphs, lists, and general text content

    **Options:**
    - Google Fonts library
    - Web-safe fallback fonts
    - Custom font URLs (advanced)

    **Popular choices:** Open Sans, Lato, Roboto, Inter, Source Sans Pro

    <Tip>
    Prioritize readability over style for body text
    </Tip>
  </Accordion>
</AccordionGroup>

<Warning>
**Email Client Compatibility:** Not all email clients support custom fonts. Migma automatically includes fallback fonts (Arial, Helvetica, sans-serif) for maximum compatibility.
</Warning>

### Set Brand Voice & Personality

Define how Migma should write email copy in your brand's voice.

**Voice Configuration Options:**

<Tabs>
  <Tab title="Brand Personality">
    Describe your brand's personality in a few sentences:

    **Example:**
    ```
    We're a friendly, innovative SaaS company that makes
    complex technology simple. We communicate in a warm,
    approachable tone while maintaining professionalism.
    We use humor sparingly and always focus on customer value.
    ```
  </Tab>

  <Tab title="Writing Style">
    Specify writing preferences:

    - **Tone:** Professional, Casual, Friendly, Authoritative, Playful
    - **Formality:** Formal, Semi-formal, Informal
    - **Person:** First person (we/our), Second person (you/your), Third person
    - **Sentence length:** Short and punchy, Medium, Long and detailed
  </Tab>

  <Tab title="Do's and Don'ts">
    **What to include:**
    - Words and phrases you commonly use
    - Your unique terminology
    - Preferred call-to-action language

    **What to avoid:**
    - Words that don't fit your brand
    - Competitor terminology
    - Overly promotional language
  </Tab>

  <Tab title="Sample Copy">
    Paste example copy from your website or marketing materials:

    - Homepage copy
    - Product descriptions
    - Email examples
    - Blog posts

    **Why:** Helps AI learn your specific writing patterns
  </Tab>
</Tabs>

### Add Style Guidelines

Provide additional style guidelines for email design.

**What to Include:**

```markdown
- Layout preferences (single column, hero images, minimal)
- Button styles (rounded, square, pill-shaped)
- Image usage guidelines
- Spacing preferences
- Email structure patterns
- Section layouts
```

**Example Style Guidelines:**

```
Our emails should feel modern and clean with plenty of white space.
We prefer single-column layouts with large, bold headlines. CTAs should
be prominent with rounded corners. We always include a hero image at
the top. Avoid cluttered designs - keep it simple and scannable.
```

### Add Custom CSS

For advanced users: Add custom CSS to fine-tune email styling.

**Use Cases:**
- Custom button styles
- Specific spacing adjustments
- Unique design patterns
- Brand-specific components

**Example:**

```css
/* Custom button styles */
.cta-button {
  border-radius: 8px;
  padding: 16px 32px;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 1px;
}

/* Brand-specific header */
.email-header {
  background: linear-gradient(135deg, #6D28D9 0%, #A78BFA 100%);
  padding: 40px 20px;
}
```

<Warning>
**Email Client Compatibility:** Not all CSS is supported in all email clients. Test your custom CSS thoroughly.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Brand import failed" icon="triangle-exclamation">
    **Common causes:**
    - Website is behind authentication/login
    - Website blocks crawlers (robots.txt)
    - Website has no accessible styling (headless CMS)

    **Solution:** Use manual configuration or contact support
  </Accordion>

  <Accordion title="Wrong colors extracted" icon="palette">
    **Why:** AI might pick colors from images or background elements

    **Solution:** Manually override the colors in project settings
  </Accordion>

  <Accordion title="Logos not showing in emails" icon="image">
    **Common causes:**
    - Logo file too large (over 2MB)
    - Logo URL is private/inaccessible
    - Logo format not supported

    **Solution:** Re-upload in PNG or SVG format, ensure public URL
  </Accordion>

  <Accordion title="Fonts don't match my website" icon="font">
    **Why:** Not all web fonts work in email clients

    **Solution:** Choose the closest Google Font alternative or use web-safe fallbacks
  </Accordion>
</AccordionGroup>
